# The Zora Files

> Every episode is a scored attempt at the perfect sunrise — the desert doesn't care about your plan.

This repository is the operational hub for **The Zora Files** channel and the **Finding Zora** series. Everything from brand guidelines to episode planning to the website lives here.

## Directory map

```
the-zora-files/
├── ZORA_PROJECT.md          ← Master context document (source of truth for all decisions)
├── PLAN.md                  ← Current project plan and task tracker
├── README.md                ← You are here
│
├── brand/                   ← Brand identity assets and guidelines
│   ├── colors/              ← Color palette swatches, exports
│   ├── typography/          ← Font files, usage notes
│   ├── logos/               ← Logo variations (dark, light, icon, wordmark)
│   └── templates/           ← Thumbnail templates, score card templates, lower thirds
│
├── episodes/                ← Episode planning, scripts, and scoring
│   ├── _templates/          ← Blank episode planning template
│   └── season-01/           ← One folder per episode (S01E01, S01E02, etc.)
│
├── discoveries/             ← Discovery Log data — species, features, cultural sites
│
├── production/              ← Production workflow resources
│   ├── checklists/          ← Pre-shoot, on-location, post-production checklists
│   ├── luts/                ← DaVinci Resolve color grade LUTs
│   └── davinci-templates/   ← Resolve project templates, motion graphics
│
├── website/                 ← thezorafiles.com — Next.js project (initialized when ready)
│
├── social/                  ← Platform-specific content and assets
│   ├── youtube/             ← Channel art, banner, descriptions, SEO
│   ├── instagram/           ← Bio, highlight covers, post templates
│   ├── tiktok/              ← Bio, content notes
│   └── x-twitter/           ← Defensive registration notes
│
└── planning/                ← Strategic plans, research, location scouting
```
