# Post-production checklist

_After every shoot, work through this pipeline._

## Ingest
- [ ] Import all clips via Hedge to organized folder
- [ ] Folder naming: `S01E01-the-benchmark/raw/[device-name]/`
- [ ] Verify all files transferred — check clip counts against shot log
- [ ] Back up raw footage to secondary drive

## Edit
- [ ] Create DaVinci Resolve project from episode template
- [ ] Sync all cameras via timestamp (clocks were synced on location)
- [ ] Rough cut — follow the episode structure:
  - Cold open (30-60 sec)
  - The approach
  - Discovery window
  - The sunrise
  - The verdict (60-90 sec max)
- [ ] Apply color grade LUT
  - Warm amber for golden hour footage
  - Cool blue for pre-dawn / blue hour footage
- [ ] Add Eos Index score reveal graphic
- [ ] Add lower thirds for locations and species
- [ ] Export to Descript for transcript-based talk segment editing

## Score
- [ ] Complete the Eos Index scoring in the episode's `score.md`
- [ ] Update the leaderboard if this episode places
- [ ] Update the Discovery Log with all new species/features

## Shorts
- [ ] Export long-form to OpusClip for auto-clipping
- [ ] Select and polish 2-3 best clips in CapCut
- [ ] Add captions and format for 9:16

## Upload
- [ ] YouTube — upload long-form first
  - Title, description, tags, thumbnail
  - End screen + cards
  - Schedule or publish
- [ ] TikTok — upload 2-3 shorts
- [ ] Instagram Reels — cross-post shorts
- [ ] Update thezorafiles.com with new episode data
