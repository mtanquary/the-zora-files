# On-location checklist

_Run through on arrival at the shoot location._

## Arrival
- [ ] Sync all device clocks via their apps
- [ ] Set up GorillaPod in locked timelapse position — do not move it
- [ ] Start Merlin Bird ID passive sound logging
- [ ] Start GoPro always-on wide POV
- [ ] Open iNaturalist — ready for photo IDs

## Camera settings (Filmic Pro)
- [ ] Manual white balance: 4000K
- [ ] Frame rate: 24fps
- [ ] ISO: as low as possible
- [ ] Lock all settings before sunrise begins

## During the shoot
- [ ] Film establishing wide shot — trailhead/vehicle
- [ ] Film approach — trail, road conditions, gear visible naturally
- [ ] Discovery window — dark-to-dawn wildlife and feature hunting
- [ ] Photo-ID every wildlife/plant encounter with iNaturalist
- [ ] The sunrise — camera locked, don't touch the timelapse
- [ ] Score the sunrise live on camera (rough, refine in post)

## Before leaving
- [ ] Review timelapse — did it run uninterrupted?
- [ ] Review audio levels from DJI Mic
- [ ] Note any species heard but not seen (Merlin log)
- [ ] Take a final wide establishing shot of the location in daylight
