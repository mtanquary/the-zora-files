# Pre-shoot checklist

_Complete the night before every shoot._

## Planning
- [ ] Confirm sunrise time — [timeanddate.com](https://www.timeanddate.com)
- [ ] Check weather forecast — note cloud cover (good for scoring) vs. clear
- [ ] Check moon phase — dark moon = better pre-dawn shots
- [ ] Review eBird recent sightings for the location
- [ ] Review PhotoPills for sun position at the specific GPS coordinates
- [ ] Confirm park/trailhead open time and fees

## Gear
- [ ] Samsung S25 Ultra — charged, storage cleared
- [ ] GoPro — charged, storage cleared (when acquired)
- [ ] DJI Mic 2 — charged, paired (when acquired)
- [ ] GorillaPod + phone mount — packed (when acquired)
- [ ] Drone — charged, preflight check (when acquired)
- [ ] Backup battery pack — charged

## Apps
- [ ] Filmic Pro — settings verified (manual WB 4000K, 24fps)
- [ ] Merlin Bird ID — ready, location set
- [ ] iNaturalist — logged in, location enabled
- [ ] eBird — checklist ready for the location

## Departure
- [ ] Know your drive time — arrive 45 min before sunrise
- [ ] Cash/card for park fees
- [ ] Water, layers, headlamp
