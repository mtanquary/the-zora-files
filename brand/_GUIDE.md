# Brand — guide

This folder holds all visual identity assets for The Zora Files.

## Subfolders

### colors/
Store color palette reference files here — exported swatches (.ase, .clr), social media palette images, or quick-reference PNGs. The canonical palette is defined in [ZORA_PROJECT.md](../ZORA_PROJECT.md) under "Color Palette."

| Name | Hex | Role |
|------|-----|------|
| Pre-dawn | `#0D0F14` | Dark backgrounds |
| Zora amber | `#F0A500` | Primary accent |
| Sunrise orange | `#E8520A` | Secondary accent, conditions bar |
| Eos teal | `#1D9E75` | Eos Index only |
| Dawn mist | `#C8D4E0` | Body text on dark |

### typography/
Font files (.otf, .ttf, .woff2) and a usage note explaining which font is used where. Final font selections are pending — add them here once chosen.

### logos/
All logo variations. Naming convention:
- `zora-logo-dark.png` — for use on dark backgrounds
- `zora-logo-light.png` — for use on light backgrounds
- `zora-icon.png` — square icon (profile pics, favicons)
- `zora-wordmark.png` — text-only version
- Keep source files (.ai, .fig, .svg) alongside exports

### templates/
Reusable design templates:
- **Thumbnail template** — consistent layout for YouTube thumbnails
- **Eos score card** — the on-screen scoring breakdown graphic
- **Lower third** — name/location overlay for video
- **Discovery card** — the "new species unlocked" graphic

## Rules
- Amber is the primary accent everywhere
- Teal is reserved exclusively for Eos Index elements
- Never use pure black (#000000) or pure white (#ffffff)
- All copy in sentence case — never ALL CAPS or Title Case in UI
