# Episodes — guide

This folder contains all episode planning, scripts, notes, and scoring for every episode.

## Structure

```
episodes/
├── _templates/
│   └── episode-template.md    ← Copy this to start planning a new episode
└── season-01/
    ├── S01E01-the-benchmark/
    │   ├── plan.md            ← Location, logistics, targets, shot list, full scoring
    │   ├── discoveries.md     ← Species/features found this episode
    │   └── notes.md           ← Field notes, things that happened, lessons
    ├── S01E02-the-habitat-flip/
    └── S01E03-the-time-capsule/
```

## How to use

1. **Copy** `_templates/episode-template.md` into the season folder
2. **Rename** the folder using the convention: `S01E01-short-title`
3. **Fill in** the plan before the shoot
4. **Score** the episode after filming — both Eos Index and full Zora Score
5. **Log discoveries** — these feed into the Discovery Log and Zora Score

## Naming convention

Folder: `S01E01-the-benchmark`
Title format: `S01E01 — "The Benchmark" — Lost Dutchman SP`

## Dual scoring system

Every episode gets two scores:

### Eos Index — sunrise quality (0–100)
| Category | Max | What it measures |
|----------|-----|-----------------|
| Sky | 50 | Color intensity (20) + Cloud engagement (15) + Horizon definition (15) |
| Setting | 30 | Foreground composition (15) + Location uniqueness (15) |
| Conditions | 20 | Effort to reach (10) + Weather/environmental challenge (10) |

### Zora Score (no fixed ceiling)
Eos Index (0-100) + Effort Rating (0-40, non-linear) + Discovery Points (variable)

See [ZORA_PROJECT.md](../ZORA_PROJECT.md) for full point tables.
